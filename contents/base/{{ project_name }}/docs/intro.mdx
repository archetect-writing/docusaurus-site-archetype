## Getting Started

This site is built with [Docusaurus](https://docusaurus.io/) — a modern
static documentation generator.

### Prerequisites

- [Node.js](https://nodejs.org/) version 20 or above

### Run locally

```bash
npm install
npm start
```

The dev server reloads automatically when you edit files.
